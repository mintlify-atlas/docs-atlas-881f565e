  <p className="dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with dotly in just a few minutes
  </p>

  <Steps>
    <Step title="Install dotly">
      Run the installer script to set up dotly and create your dotfiles structure:
      
      ```bash curl
      bash <(curl -s https://raw.githubusercontent.com/CodelyTV/dotly/HEAD/installer)
      ```
      
      Or with wget:
      
      ```bash wget
      bash <(wget -qO- https://raw.githubusercontent.com/CodelyTV/dotly/HEAD/installer)
      ```
    </Step>
    
    <Step title="Commit your dotfiles">
      Create a new GitHub repository named `dotfiles` and push your initial configuration:
      
      ```bash
      cd ~/.dotfiles
      git remote add origin YOUR_DOTFILES_REPO_URL
      git add -A
      git commit -m "Initial commit"
      git push origin main
      ```
    </Step>
    
    <Step title="Use the dot command">
      Start using the `dot` command to manage scripts and configurations:
      
      ```bash
      dot
      ```
      
      This opens an interactive fzf menu with all available scripts. You can also run specific scripts:
      
      ```bash
      dot package add neovim
      dot git pretty-log
      ```
    </Step>
  </Steps>
</div>
